# Process Monitor

[![Version](https://img.shields.io/badge/version-v0.0.6-blue.svg)](https://github.com/mk89-sudo/Process_monitor/releases)
[![Python](https://img.shields.io/badge/python-3.11+-green.svg)](https://www.python.org/downloads/)
[![License](https://img.shields.io/badge/license-MIT-orange.svg)](LICENSE)

**Process Monitor** — это приложение для Windows (и других ОС) с графическим интерфейсом на `tkinter`, которое отслеживает запущенные процессы и отправляет уведомления через сервис [ntfy.sh](https://ntfy.sh) при их старте или остановке.

Программа позволяет отслеживать все экземпляры процессов по имени (например, все окна `chrome.exe`), имеет встроенную систему автоматического обновления, темную тему и возможность сворачивания в системный трей.

---

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🔍 **Мониторинг процессов** | Отслеживание всех экземпляров процесса по имени |
| 📢 **Уведомления** | Отправка через ntfy.sh при старте/остановке процессов |
| 🎨 **Темная тема** | Полностью темный интерфейс для комфортной работы |
| 📌 **Системный трей** | Сворачивание в трей при нажатии на крестик |
| 🔄 **Автообновление** | Проверка новой версии на GitHub (через 1 сек после запуска) |
| 📝 **Логирование** | Все события сохраняются в файл `log.txt` с точностью до миллисекунд |
| 📋 **Копирование логов** | Поддержка Ctrl+C и контекстного меню в Event Log |
| ⚙️ **Настройки** | Сохранение ntfy topic/server и списка отслеживаемых процессов |
| 🖥️ **Информация о системе** | Имя компьютера и пользователя в уведомлениях |
